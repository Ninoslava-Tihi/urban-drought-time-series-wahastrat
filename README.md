# urban-drought-time-series-wahastrat
Time series forecasting of urban drought climatic drivers using SARIMA and Holt–Winters models.
# Urban Drought Time Series Forecasting

This repository contains the processed monthly climatic dataset (2014–2020) derived from the WAHASTRAT project.

Variables:
- Air Temperature (monthly mean)
- Precipitation (monthly total)
- Soil Moisture (monthly mean)
- Wind Speed (monthly mean)

Models:
- (S)ARIMA
- Holt–Winters (ETS)

Validation methods:
- 80/20 holdout validation
- Rolling-origin cross-validation (48 folds)

This repository supports the reproducibility of the associated manuscript submitted to Sustainability (MDPI).
